## Installation
Install below packages
```bash
$ npm install
$ npm install @uifabric/styling
$ npm install office-ui-fabric-react
$ npm install axios
$ npm install lodash
$ npm install bootstrap
$ npm install react-table
```
## Development

For development, run:

```bash
$ npm start
```
